# 记负均正数

题号：HJ97
本题知识点：数组 思维

[toc]

### 题目描述

首先输入要输入的整数个数n，然后输入n个整数。输出为n个整数中负数的个数，和所有正整数的平均值，结果保留一位小数。
本题有多组输入用例。

### 输入描述

首先输入一个正整数n，
然后输入n个整数。

### 输出描述

输出负数的个数，和所有正整数的平均值。

### 限制
时间限制：C/C++ 1秒，其他语言2秒 
空间限制：C/C++ 32M，其他语言64M

### 示例

输入：
```
5
1 2 3 4 5
```

输出：
```
0 3.0
```
