# 记负均正II

题号：HJ105
本题知识点：数组 思维

[toc]

### 题目描述

从输入任意个整型数，统计其中的负数个数并求所有非负数的平均值，结果保留一位小数，如果没有非负数，则平均值为0
本题有多组输入数据，输入到文件末尾，请使用while(cin>>)读入
数据范围小于1e6

### 输入描述

输入任意个整数，每行输入一个。

### 输出描述

输出负数个数以及所有非负数的平均值

### 限制
时间限制：C/C++ 1秒，其他语言2秒 
空间限制：C/C++ 32M，其他语言64M

### 示例

输入：
```
-13
-4
-7
```

输出：
```
3
0.0
```
