# 统计每个月兔子的总数

题号：HJ37
本题知识点：查找 排序

[toc]

### 题目描述

有一只兔子，从出生后第3个月起每个月都生一只兔子，小兔子长到第三个月后每个月又生一只兔子，假如兔子都不死，问每个月的兔子总数为多少？

本题有多组数据。

### 输入描述

输入int型表示month

### 输出描述

输出兔子总数int型

### 限制
时间限制：C/C++ 1秒，其他语言2秒 
空间限制：C/C++ 32M，其他语言64M

### 示例

输入：
```
9
```

输出：
```
34
```
