# 查找组成一个偶数最接近的两个素数

题号：HJ60
本题知识点：查找 数学 穷举

[toc]

### 题目描述

任意一个偶数（大于2）都可以由2个素数组成，组成偶数的2个素数有很多种情况，本题目要求输出组成指定偶数的两个素数差值最小的素数对。
本题含有多组样例输入。

### 输入描述

输入一个偶数

### 输出描述

输出两个素数

### 限制
时间限制：C/C++ 1秒，其他语言2秒 
空间限制：C/C++ 32M，其他语言64M

### 示例

输入：
```
20
```

输出：
```
7
13
```
